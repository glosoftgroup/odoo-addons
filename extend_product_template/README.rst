This odoo addon extend a product.template table.
It can be used if user wants to add a new field to products table

installation 
===================================--========
Ensure you have started odoo server and you are in debugger mode

1. Copy extend product template addon on you odoo addon folder
2. Click Apps menu, update app list and search extend product template app
3. Install the addon
4. Navigate sales module and click create product.

================================================
On successful installation, you will notice that an new page with extend field 
was added
